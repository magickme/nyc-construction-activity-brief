NYC Construction Activity Brief

NYC DOB permit CSV sample for construction activity review.

The first issue package includes a public CSV preview and Markdown brief built from selected NYC DOB NOW records for a specific weekly window.

Who it helps

Small construction vendors and subcontractors that need a spreadsheet-friendly way to review current public permit activity.

What you get

Work type, ZIP, borough, issued date, status, cost bucket, source link, permit ID, filing number, and short description.

Current sample

The public CSV preview includes 192 rows across seven selected work types and five selected ZIP codes.

Use case

Use the CSV page if your main need is filtering and sorting. It is built for spreadsheet review, not CRM import or automated contact scraping.
